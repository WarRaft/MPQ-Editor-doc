# open

<snippet id="main">

Opens an existing MPQ. If the file doesn't exist, the function fails.

Syntax
:
```
open MpqFile [ListFile]
```

| MpqFile  | Name of the MPQ to be used for this operation.     |
|----------|----------------------------------------------------|
| ListFile | Name of the listfile to be used on open operation. |
{style=none}

</snippet>
