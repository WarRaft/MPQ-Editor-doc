# rename

<snippet id="main">

Renames a file within MPQ.

Syntax
:
```
rename MpqFile OldFileName NewFileName
```

| MpqFile     | Name of the MPQ to be used for this operation. |
|-------------|------------------------------------------------|
| OldFileName | Name of file to be renamed.                    |
| NewFileName | New name of the file                           |
{style=none}

</snippet>
