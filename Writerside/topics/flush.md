# flush

<snippet id="main">

Compacts the MPQ, removing all empty space.

Syntax
:
```
flush MpqFile [ListFile]
```

| MpqFile  | Name of the MPQ to be used for this operation.         |
|----------|--------------------------------------------------------|
| ListFile | Listfile name. If none, the internal listfile is used. |
{style=none}

</snippet>
